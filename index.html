Introducción a Git y GitHub

¿Qué es Git?

Git es un sistema de control de versiones que permite guardar el historial de cambios de un proyecto, facilitando la colaboración entre múltiples desarrolladores.

¿Qué es GitHub?

GitHub es una plataforma en línea que permite almacenar repositorios Git en la nube, compartir código y colaborar en proyectos de manera sencilla.

¿Qué es un repositorio?

Un repositorio es un contenedor donde se almacena el código de un proyecto junto con todo su historial de cambios y versiones.

Commits, ramas, push y pull

Commits: Son registros de cambios que se realizan en el proyecto, cada uno con un mensaje descriptivo.

Ramas: Permiten trabajar en diferentes versiones del proyecto sin afectar el código principal.

Push: Envía tus cambios locales al repositorio remoto.

Pull: Descarga los cambios más recientes del repositorio remoto a tu entorno local.
